# 📝 Lista de Itens em React

Este projeto é uma aplicação simples em **React** que permite adicionar e remover itens de uma lista de forma dinâmica.

## 🚀 Funcionalidades
- Adicionar itens à lista.
- Remover itens da lista.
- Limpeza automática do input após adicionar um item.
- Interface simples e intuitiva.

## 💻 Tecnologias Utilizadas
- React
- useState
- JSX
