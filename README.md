# 📋 Lista de Itens (CRUD com React)

Este projeto é uma aplicação simples desenvolvida em **React** que permite gerenciar uma lista de itens com as operações básicas de um CRUD:

* ✅ Adicionar itens
* ✏️ Atualizar (editar) itens
* ❌ Remover itens
* 📄 Listar itens

---

## 🚀 Tecnologias utilizadas

* React (useState)
* JavaScript
* HTML + CSS

---

## 📸 Funcionalidades

### ➕ Adicionar item

O usuário digita um texto no campo e clica em **Adicionar** para inserir um novo item na lista.

### ✏️ Atualizar item

Ao clicar no botão **Editar**, o item é carregado no campo de texto, permitindo sua alteração.
Depois, basta clicar em **Atualizar** para salvar.

### ❌ Remover item

Cada item possui um botão **Remover**, que exclui o item da lista.

---

## 🧠 Como funciona

O sistema utiliza o **useState** para controlar:

* A lista de itens (`Itens`)
* O texto digitado (`texto`)
* O índice do item em edição (`indiceAtualizar`)
